# Project Overview

Веб-приложение предназначено для демонстрации учебных и практических работ.
Сайт используется как витрина выполненных заданий и проектов с упором на простую структуру и наглядный просмотр материалов.

Проект реализован как **SPA на React**.
Данные загружаются асинхронно из **JSON-файлов**, расположенных в `public/data`.

Все материалы размещаются на одном хостинге.

## Project Goals

- Централизованное хранение учебных материалов
- Удобная навигация по типам работ
- Простая и предсказуемая структура данных
- Возможность обновлять данные без пересборки приложения

## Key principles

- Простота
- Отсутствие бекенда
- Данные обновляются без пересборки
- Чёткая структура данных

## Technology Stack

- React ^19
- TypeScript ~5
- React Router DOM ^7
- Tailwind CSS ^4
- Vite + Rolldown
