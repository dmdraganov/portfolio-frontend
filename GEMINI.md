# Project Overview

This is a portfolio web application designed to showcase educational and practical work. It functions as a single-page application (SPA) built with React and TypeScript, displaying projects and assignments loaded from local JSON files.

## Documentation

For more detailed information about the project, please see the documentation files in the `docs` directory:

- [Architecture](./docs/architecture.md)
- [Constraints](./docs/constraints.md)
- [Data](./docs/data.md)
- [Routes](./docs/routes.md)
- [UI Guide](./docs/ui-guide.md)

## Technology Stack

- **Framework**: React
- **Language**: TypeScript
- **Build Tool**: Vite (with Rolldown)
- **Routing**: React Router DOM
- **Styling**: Tailwind CSS

## Building and Running

To work with this project, you need to have Node.js and npm installed.

### Key Commands

- **Install dependencies**:

  ```bash
  npm install
  ```

- **Run the development server**:

  ```bash
  npm run dev
  ```

- **Build for production**:

  ```bash
  npm run build
  ```

- **Lint the code**:

  ```bash
  npm run lint
  ```

- **Format the code**:
  ```bash
  npm run format
  ```
